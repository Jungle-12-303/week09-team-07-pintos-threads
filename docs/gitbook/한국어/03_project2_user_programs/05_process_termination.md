# 05_process_termination

## 프로세스 종료 메시지

**프로세스 종료 메시지를 출력하세요**

사용자 프로세스가 `exit`를 호출했든 다른 이유로 종료되었든, **프로세스의 이름과 종료 코드를 출력하세요**. 형식은 아래 `printf`가 출력하는 형태와 같아야 합니다.

```c
printf ("%s: exit(%d)\n", ...);
```

출력할 이름은 `fork()`에 전달했던 전체 이름이어야 합니다. 사용자 프로세스가 아닌 커널 스레드가 종료할 때나 halt 시스템 콜이 호출될 때는 이 메시지를 출력하지 마세요. 프로세스 로드에 실패했을 때는 이 메시지를 출력할지 여부가 선택 사항입니다.

이 외에는, 기본 Pintos가 이미 출력하는 것 외의 다른 메시지는 출력하지 마세요. 디버깅할 때는 추가 메시지가 유용할 수 있지만, 그런 출력은 채점 스크립트를 혼란스럽게 만들어 점수를 떨어뜨립니다.
